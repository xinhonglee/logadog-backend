## LogADog Backend

### Prerequisites

-   Node.js > 8.12

-   MongoDB

### Installation

-   make environment


    $ cp .env.example .env

-   install packages


    $ npm install

### Run Project

-   Db Migration


    $ npm run fresh

-   dev environment


    $ npm run dev

-   production environment


    $ npm run start

### Testing Project

    npm run test
